In this folder you fill find two files after Programm execution:
-DriverSVH_Debug.log  
This log files saves the DEBUG messages (and above) of the svh library and is usefull if you encounter a problem and want to dig deeper into it

-DriverSVH_Trace.log
This log file saves the TRACE messages (and above) of the svh library. The ouput here is very verbose and is mainly intended to be sent to the
package maintainer in case you run into serious trouble. It should provide a very detailed insight into whats happening.
